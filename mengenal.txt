Github adalah sebuah software yang berisi fitur Git didalamnya yang berguna untuk mengelola dan melacak file atau projek
Github membutuhkan koneksi internet sedang git tidak perlu
Kegunaan github adalah sebagai berikut ;
- Mengelola file, projek atau berkas dengan mudah karena banyak fiturnya
- Bisa untuk berkolaborasi dengan yang lainnya terkait sebuah projek
